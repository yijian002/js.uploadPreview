# Upload Preview

Upload image preview, Javascript plugin.


### Code examples
``` js
window.uploadPreview({
    label_field: 'callback-upload',
    preview: 'preview-id'
});
```

### AMD require.js
``` js
require(['uploadPreview'], function(uploadPreview) {
    // ...
});
```
